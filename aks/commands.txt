create azure container registry after you logged in and created resources group
1. az acr create --resource-group resourceGroupName --name acrName --sku Basic 
----Enable the access key admin account--------------------------------------
2. az acr update acrName --admin-enabled true
3. az acr login -n MyRegistry --expose-token

Create a secrete that docker will use to authenticate with azure container registry.
kubectl create secret docker-registry secretename[eg: ac-secret] --docker-server=acrserverUri[eg:acrName.azurecr.io]
 --docker-username=acrName --docker-password=acr-passcode-after-enabled[eg:Ssasdfd5ErHGFAXtF58cNnm48y93BaE=uvEjoM]

 az acr repository show-tags --name registryName --repository imageName --output table # get tags of specified image
 az acr repository list --name registryName --output table # get all the images in the container
